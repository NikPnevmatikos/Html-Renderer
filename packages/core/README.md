# @html-renderer/core

Pure-JavaScript HTML renderer for React Native. Fabric/Expo compatible.

See the [root README](../../README.md) for full documentation.

## Install

```bash
npm install @html-renderer/core
```

## Usage

```tsx
import { HtmlRenderer } from '@html-renderer/core';

<HtmlRenderer html="<p>Hello <strong>world</strong></p>" />;
```
