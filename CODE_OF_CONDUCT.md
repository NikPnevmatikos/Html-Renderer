# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project, you agree to uphold this code. Please report unacceptable behavior to the maintainers by opening a confidential GitHub security advisory (see [SECURITY.md](./SECURITY.md)) or emailing the address listed there.
